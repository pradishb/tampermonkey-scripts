# Tampermonkey Scripts

A collection of Tampermonkey userscripts for blocking distractions and staying focused.

## Scripts

### Distraction Free YouTube

Reduces distractions on YouTube by:

- Hiding the homepage feed
- Hiding "watch next" suggestions in the sidebar
- Hiding video suggestions that appear at the end of a video
- Redirecting Shorts to the regular video player (`/watch?v=...`)

### Distraction Free Instagram

Keeps Instagram limited to direct messages by:

- Redirecting the home feed, Reels, and Explore pages to the DM inbox

### Site Blocker

Blocks distracting websites entirely by redirecting them to `about:blank`.

Currently blocks:

- 9gag.com
